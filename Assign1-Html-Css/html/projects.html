Projects


Mobile Store System:

This system is built to manage almost all activities of Mobile Store, it is a Desktop application. It easily helps in managing data of customer, mobile and accessories, expenses of store, supplier, employee and their salaries, etc. Proper validation for each and every field ensures only clean and consistent data is saved. This system can calculate bill with GST and discount, generate and print bill, generate filtered reports, effectively search for products, etc. This system is password protected, with option to change password.



Two-Wheeler Showroom:

This system has wonderful UI and it is a desktop application built to handle operations of Two-Wheeler Showroom. It is built to maintain details of bikes, accessories and its availability, along with insurance. It is also used to manage data of employee, customer, and showroom expenses. Filtered and customized Reports can be generated, along with booking/buying bikes and accessories. Two copies of invoice are generated one for showroom and one for customer. This system takes minimum input with validation, which makes is simpler to understand and use. User can easily change password if he forgets.



E-Commerce Portal:

E-commerce Portal is a Web Application for online Furniture Store. Users can easily log on to the web site to buy wide range of household and office furniture. Products are categorized for the convenience of user. User can add products to cart, track order, manage profile, etc. Whereas Admin/Site Managers can manage database of products, update order status, generate sales report, etc. Passwords are encrypted to avoid unauthorized access.
